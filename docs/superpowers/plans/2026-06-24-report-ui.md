# AgentDiff Report UI + Real Demo — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a YC-grade report/dashboard UI that renders AgentDiff's real `compare`-run data, proven by GIFs of the actual UI on a real Ollama-backed demo run, plus README + GitHub profile.

**Architecture:** The AgentDiff engine is untouched. A new read-only `report_payload.build()` assembles the full UI payload from the artifacts a run already wrote to `agentdiff.sqlite`; `dashboard.py` injects it as `window.__AGENTDIFF__`. A small additive Ollama branch on `LLMClient` lets the judge/explainer target a local model. The frontend (React/Vite/TS/Tailwind/shadcn) renders five sections with the before/after agent graph as the hero. A scaffolded research-assistant sample app (the agent-under-test) produces a real regression for the demo.

**Tech Stack:** Python 3.10+ (pydantic v2, click, sqlite3), React 18 + Vite 6 + TypeScript 5 + Tailwind 3 + shadcn/ui + `@xyflow/react`, Ollama (`llama3.1:8b`), pytest, ffmpeg/gifski for GIFs.

## Data Contract (single source of truth — referenced by Tasks 2, 3, 7)

`report_payload.build(report_dir: Path) -> dict` returns:

```jsonc
{
  "meta":        { "baseline_ref": str|null, "candidate_ref": str|null,
                   "samples_per_case": int|str|null, "timestamp": str|null,
                   "smoke_mode": bool },
  "runQuality":  { "baseline_trajectories": int|null, "candidate_trajectories": int|null,
                   "baseline_failed": int|null, "candidate_failed": int|null,
                   "max_failure_rate": float|null, "thresholds": object|null },
  "graph":       { /* exact AgentGraph.model_dump() from graph_model.build */ },
  "comparison":  { "overall_verdict": "pass"|"warn"|"fail",
                   "test_case_comparisons": [ {
                     "test_case_id": str, "overall_verdict": str,
                     "behavioral_overlap": float|null,
                     "agent_invocation_deltas": [ /* AgentInvocationDelta dumps */ ],
                     "tool_usage_deltas": [ /* ToolUsageDelta dumps */ ] } ] } | null,
  "outputEvals": [ { "test_case_id": str, "output_kind": str,
                     "semantic_similarity": float|null, "structural_similarity": float|null,
                     "length_ratio": float|null, "judge_score": float|null,
                     "verdict": str, "notes": [str] } ],
  "attribution": { "attributions": [ {
                     "agent_name": str, "delta_summary": str, "verdict": str,
                     "primary": { "target_path": str, "rule": str, "weight": float,
                                  "reason": str, "hunk": str|null } | null,
                     "alternatives": [ { "target_path": str, "rule": str, "weight": float } ],
                     "explanation": str|null } ] } | null,
  "trajectories": {
     "baseline":  [ { "trajectory_id": str, "test_case_id": str, "status": str,
                      "final_output": str|null, "total_tokens": int, "total_latency_ms": int,
                      "timeline": [ TimelineEvent ] } ],
     "candidate": [ /* same shape */ ] }
}
```

`TimelineEvent` (projected from `events` rows — only the fields the UI needs):

```jsonc
{ "seq": int, "kind": str /* event_type */, "inferred_agent": str|null,
  "provider": str|null, "model": str|null, "latency_ms": int|null,
  "usage": object|null, "tool_name": str|null,
  "request_preview": str|null, "response_preview": str|null }
```

Field sources (all already in the DB/artifacts — nothing recomputed):
- `meta`/`runQuality` ← `metadata.json` keys written by `cli/compare.py` (`baseline_ref`, `candidate_ref`, `samples_per_case`, `timestamp`, `smoke_mode`, `baseline_trajectories`, `candidate_trajectories`, `baseline_failed`, `candidate_failed`, `max_failure_rate`, `thresholds`).
- `graph` ← `graph_model.build(read_artifact("comparison"), read_artifact("attribution"), load_trajectory_set_from_sqlite("baseline"), load_trajectory_set_from_sqlite("candidate"))`.
- `comparison` ← `read_artifact(db, "comparison")` (already a dict).
- `outputEvals` ← `read_artifact(db, "output_evals")` (already a list of dicts).
- `attribution` ← `read_artifact(db, "attribution")` (already a dict or None).
- `trajectories` ← `load_trajectory_set_from_sqlite(db, side)`; project each `Trajectory` via `_project_timeline`.

`_project_timeline(traj)` maps each `event` (a pydantic `Event`) to a `TimelineEvent`:
- `seq` = `getattr(ev, "sequence", 0)`; `kind` = `ev.event_type`; `inferred_agent` = `getattr(ev, "inferred_agent", None)`.
- For `LLMRequestEvent`/`LLMResponseEvent`: `provider`/`model` from `ev.canonical.provider`/`ev.canonical.model`; `latency_ms` = `getattr(ev, "latency_ms", None)`; `usage` = `ev.canonical.usage or None`; `request_preview` = first user message text (truncated 280 chars); `response_preview` = `ev.canonical.response_text` (truncated 280) for responses.
- For `MCPToolInvokedEvent`/`LocalToolInvokedEvent`: `tool_name` = `ev.tool_name`; `request_preview` = `json.dumps(ev.arguments)[:280]`.
- Everything else: leave the type-specific fields `None`.

import json
from pathlib import Path
from uuid import uuid4

from agentdiff import report_payload, storage
from agentdiff.compare import (
    AgentInvocationDelta, ComparisonResult, TestCaseComparison, ToolUsageDelta,
)
from agentdiff.capture.events import CanonicalLLMCall, CallSite, LLMResponseEvent
from agentdiff.trajectory import Trajectory, TrajectorySet


def _llm_event(seq, agent, model, text):
    return LLMResponseEvent(
        sequence=seq, latency_ms=42, call_id=uuid4(),
        canonical=CanonicalLLMCall(provider="openai", model=model,
                                   response_text=text, usage={"input_tokens": 5, "output_tokens": 7}),
        captured_by="http_shim",
    )


def _traj(tag, tcid, agent, model, text):
    t = Trajectory(test_case_id=tcid, version_tag=tag, input={"query": "q"},
                   final_output=text, total_tokens=12, total_latency_ms=42,
                   events=[_llm_event(0, agent, model, text)])
    # tag the event's inferred_agent (set post-construction; field is optional)
    t.events[0].inferred_agent = agent
    return t


def _write_run(tmp_path: Path) -> Path:

        overall_verdict="fail",
        test_case_comparisons=[TestCaseComparison(
            test_case_id="basic", overall_verdict="fail", behavioral_overlap=0.5,
            agent_invocation_deltas=[AgentInvocationDelta(
                agent_name="fact_checker", function="fact_checker_agent",
                baseline_rate=1.0, candidate_rate=0.0, delta=-1.0,
                baseline_count=2, candidate_count=0, baseline_total=2, candidate_total=2,
                p_value=0.01, significant=True, verdict="fail")],
            tool_usage_deltas=[ToolUsageDelta(
                tool_name="web_search", baseline_avg=1.0, candidate_avg=2.0, delta=1.0,
                p_value=0.02, significant=True, verdict="fail")],
        )],

    baseline = TrajectorySet(version_tag="baseline",

```python
"""Read-only assembly of the full dashboard payload from a run's artifacts.

Surfaces the data a ``compare`` run already wrote to ``agentdiff.sqlite`` (plus
``metadata.json``) as one JSON-serializable dict for the React UI. Computes
nothing new — every value comes from an existing artifact.
"""
import json
from pathlib import Path
from typing import Any

from agentdiff.capture.events import (
    LLMRequestEvent, LLMResponseEvent, LocalToolInvokedEvent, MCPToolInvokedEvent,
)
from agentdiff.dashboard import summarize_report
from agentdiff.storage import load_trajectory_set_from_sqlite, read_artifact
from agentdiff.trajectory import Trajectory

_PREVIEW = 280


def build(report_dir: Path) -> dict[str, Any]:
    report_dir = Path(report_dir)
    metadata_path = report_dir / "metadata.json"
    metadata: dict[str, Any] = {}
    if metadata_path.exists():
        metadata = json.loads(metadata_path.read_text(encoding="utf-8"))
    db_path = Path(metadata.get("sqlite_store", report_dir / "agentdiff.sqlite"))

    summary = summarize_report(report_dir)  # reuses graph + meta builder
    return {
        "meta": {
            **summary["meta"],
            "smoke_mode": metadata.get("smoke_mode", False),
        },
        "runQuality": {
            "baseline_trajectories": metadata.get("baseline_trajectories"),
            "candidate_trajectories": metadata.get("candidate_trajectories"),
            "baseline_failed": metadata.get("baseline_failed"),
            "candidate_failed": metadata.get("candidate_failed"),
            "max_failure_rate": metadata.get("max_failure_rate"),
            "thresholds": metadata.get("thresholds"),
        },
        "graph": summary["graph"].model_dump(),
        "comparison": read_artifact(db_path, "comparison"),
        "outputEvals": read_artifact(db_path, "output_evals") or [],
        "attribution": read_artifact(db_path, "attribution"),
        "trajectories": {
            "baseline": _side(db_path, "baseline"),
            "candidate": _side(db_path, "candidate"),
        },
    }


def _side(db_path: Path, tag: str) -> list[dict[str, Any]]:
    tset = load_trajectory_set_from_sqlite(db_path, tag)  # type: ignore[arg-type]
    return [
        {
            "trajectory_id": str(t.run_id),
            "test_case_id": t.test_case_id,
            "status": t.status,
            "final_output": t.final_output,
            "total_tokens": t.total_tokens,
            "total_latency_ms": t.total_latency_ms,
            "timeline": _project_timeline(t),
        }
        for t in tset.trajectories
    ]


def _project_timeline(traj: Trajectory) -> list[dict[str, Any]]:
    out: list[dict[str, Any]] = []
    for ev in traj.events:
        item: dict[str, Any] = {
            "seq": getattr(ev, "sequence", 0),
            "kind": getattr(ev, "event_type", type(ev).__name__),
            "inferred_agent": getattr(ev, "inferred_agent", None),
            "provider": None, "model": None, "latency_ms": getattr(ev, "latency_ms", None),
            "usage": None, "tool_name": None,
            "request_preview": None, "response_preview": None,
        }
        if isinstance(ev, (LLMRequestEvent, LLMResponseEvent)):
            c = ev.canonical
            item["provider"] = c.provider
            item["model"] = c.model
            item["usage"] = c.usage or None
            item["request_preview"] = _first_user_text(c.messages)
            if isinstance(ev, LLMResponseEvent) and c.response_text:
                item["response_preview"] = c.response_text[:_PREVIEW]
        elif isinstance(ev, (MCPToolInvokedEvent, LocalToolInvokedEvent)):
            item["tool_name"] = ev.tool_name
            item["request_preview"] = json.dumps(ev.arguments, default=str)[:_PREVIEW]
        out.append(item)
    return out


def _first_user_text(messages: list[dict[str, Any]]) -> str | None:
    for m in messages:
        if m.get("role") == "user":
            content = m.get("content")
            if isinstance(content, str):
                return content[:_PREVIEW]
            if isinstance(content, list):
                for block in content:
                    if isinstance(block, dict) and block.get("type") == "text":
                        return str(block.get("text", ""))[:_PREVIEW]
    return None
```

### Task 4: Scaffold the research-assistant agent-under-test

**Files (all under `examples/research_assistant/`):**
- Create: `agents/__init__.py`, `agents/orchestrator.py`, `agents/retriever.py`, `agents/fact_checker.py`, `agents/summarizer.py`, `tools.py`, `prompts/orchestrator.md`, `runner.py`, `.agentdiff/structure.yaml`, `.agentdiff/config.yaml`, `.agentdiff/test_cases.yaml`, `.env.example`, `README.md`

**Design (the deterministic delta):**
- `orchestrator.run_research(query)`:
  1. Makes **one** Ollama LLM call to "plan" (so the orchestrator always fires).
  2. Reads a routing policy from `prompts/orchestrator.md` (a small front-matter/markered block parsed in code) → a `RoutingPolicy` dict: `{ "use_fact_checker": bool, "web_search_passes": int }`. Routing is decided in **code**, not by the model, so it's deterministic.
  3. Always calls `retriever_agent(query, passes=policy["web_search_passes"])` and `summarizer_agent(...)`. Calls `fact_checker_agent(...)` **iff** `policy["use_fact_checker"]`.
- `retriever_agent(query, passes)`: calls `web_search` tool `passes` times, then one Ollama call. (`passes` controls the tool-usage delta.)
- `fact_checker_agent`, `summarizer_agent`: one Ollama call each.
- Each sub-agent uses the `openai` SDK pointed at Ollama:
  ```python
  client = openai.OpenAI(base_url=os.environ["OPENAI_BASE_URL"], api_key=os.environ.get("OPENAI_API_KEY", "ollama"))
  client.chat.completions.create(model=os.environ.get("AGENT_MODEL", "llama3.1:8b"), messages=[...], max_tokens=200)
  ```
- `tools.py`: `@agentdiff.tool def web_search(query): ...` (returns a canned string — the tool is real to AgentDiff's capture; its body need not hit the network), `@agentdiff.tool def calculator(expr): ...`.
- **Baseline `prompts/orchestrator.md`** routing block → `use_fact_checker: true`, `web_search_passes: 1`.
- **Candidate change** (applied by `run_demo.sh`, left uncommitted) → `use_fact_checker: false`, `web_search_passes: 2`. This is the single attributable diff.
